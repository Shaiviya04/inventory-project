# 🛒 ABC Retail Mart – Inventory Report (February 2026)

## 📌 Project Overview

This project is an Excel-based data interpretation report for managing inventory of a retail store. It tracks stock movement, calculates total values, and provides useful insights.

## 📊 Project Details

* **Total Items:** 20
* **Total Stock In:** 973
* **Total Stock Out:** 404
* **Total Inventory Value:** ₹70,780
* **Highest Value Item:** ₹36,000 (Rice Bag)
* **Lowest Value Item:** ₹180 (Bread)

## 📁 Features

* Inventory tracking (Stock In & Stock Out)
* Automatic calculation of total value
* Identification of highest and lowest value items
* Well-formatted Excel table with proper structure

## 🛠 Tools Used

* Microsoft Excel

## 🎯 Purpose

The main objective of this project is to understand inventory management and apply data interpretation techniques using Excel.
